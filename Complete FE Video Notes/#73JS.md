#   Refactoring and best practices

+ Avoid 'new' keyword. Don't use let
  objname = new Object()
+ Instead define using let objname={...}
+ Refactor your code to now have 
  arrow functions
+ Use only arrow functions from 
  now unless for specific scenario
   of 'this'.
